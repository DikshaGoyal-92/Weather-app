# 🌦️ Weather App

A modern and responsive weather application built using React and OpenWeather API.

---

## 🚀 Features
- Search weather by city name
- Real-time weather data using API
- Displays temperature, humidity, wind speed
- Dynamic weather conditions (Clear, Clouds, Rain)
- Clean and responsive UI

---

## 🛠️ Tech Stack
- React (Vite)
- Axios
- OpenWeather API
- CSS

---

## 📦 Installation & Setup

```bash
npm install
npm run dev
